Projet 2 : Thermomètre de Bain

  • Action : Leds
  • Traitement : AOP
  • Energie : Lipo 2S
  • Acquisition : Capteur Température

Présentation du produit à développer

Une entreprise de produits pour nouveaux-nés souhaite élargir sa gamme de produits. Elle souhaite aujourd'hui étendre son catalogue vers une gamme d'accessoires pour le bain. C'est ainsi qu'elle entreprend de vous faire concevoir un thermomètre de bain dont les caractéristiques principales sont :

  • Détection d'une eau trop froide ( < 36,0°C )
  • Détection d'une eau trop chaude ( > 39,0°C )
  • Bonne autonomie ( > 24h de fonctionnement)
  • Prix d'acquisition faible de type « discount » ( < 20 €)

Mon ressenti

J'ai beaucoup aimé ce projet car on voit tout le produit en détail et on finit par le connaître parfaitement, même les parties réalisées par mes camarades. C'est un projet simple, pas trop poussé, mais très bien à faire pour un premier projet où nous sommes lâchés en autonomie.
